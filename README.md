# curso_python_fos
Un repositorio dentro del curso de python 
# Game project
Podrás correr el juego ejecutando las siguientes instrucciones en la terminal:

```sh
cd game
python3 main.py
```


# App project
Podrás correr el juego ejecutando las siguientes instrucciones en la terminal:

```sh
git clone
cd app
source env/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
